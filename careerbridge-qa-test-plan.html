Prerequisites

Confirm these services are running before starting any test:

npm run dev        # Next.js on localhost:3000 (or Cloudflare Tunnel URL)
npm run inngest    # Inngest dev server → http://localhost:8288
npm run email      # React Email preview → http://localhost:3001  (Section 7 only)

Keep the Inngest dashboard open at http://localhost:8288 in a second tab to monitor background jobs in real time.

Quick Smoke Test Checklist — Run Immediately Before Presentation

  • Sign in works for both personal and employer org account
  • Job board at / shows published listings
  • AI search at /ai-search returns results for a test query
  • Employer can create and publish a job listing
  • Job seeker can apply to a listing
  • Inngest dashboard shows no red/failed functions
  • <tunnel-url>/health returns { "status": "ok" }
  • Resume upload succeeds and AI summary appears within ~30 seconds
  • React Email preview loads at localhost:3001
  • No JavaScript console errors on key pages (/, /employer, /ai-search)
  • SERVER_URL in .env.local is set to the Cloudflare tunnel URL
